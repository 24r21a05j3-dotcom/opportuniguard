About OpportuniGuard

Real-Time College Opportunity Verifier System · B.Tech Project · CSE · A.Y. 2025–26

🎯Problem Statement

Students receive countless internship, hackathon, scholarship and job messages through WhatsApp and Telegram. Many are fake, expired, or contain phishing links. Students have no quick way to verify authenticity — leading to lost money, data theft, and wasted time.

⚙️How Verification Works

Each message is analyzed by Claude AI (Anthropic) using NLP reasoning. The system checks: deadline expiry, official domain validation (.edu, .gov, .ac.in), phishing link detection, suspicious keyword scanning (fees, urgency, guarantees), and duplicate detection. Output is a structured verdict with confidence score.

🏗️Tech Stack

Frontend: HTML5, CSS3, Vanilla JS · AI Engine: Claude Sonnet 4 (Anthropic API) · NLP: Regex + LLM semantic analysis · Auth: localStorage session management · Deployment: Single-file portable HTML
